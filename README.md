
# Casting.ai MVP

## How to use

```bash
git clone git@github.com:ouuyeah/casting.git
cd casting
```

Install it and run:

```bash
npm install
npm run dev
```


## Stack using it


```bash
Next.Js (For basic routes) - Then you can use whatever you want for routes and logic.
ReactJs - Main front library
Styled-jsx - CSS
StandardJs - Just to make code very clean.
npm run dev
```
